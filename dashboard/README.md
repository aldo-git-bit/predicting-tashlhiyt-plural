# Tashlhiyt Plural Patterns Dashboard

Streamlit dashboard for exploring data, computational experiments, results, and reporting.

## Running the Dashboard

### Option 1: Direct command
```bash
cd dashboard
streamlit run app.py
```

### Option 2: From project root
```bash
streamlit run dashboard/app.py
```

The dashboard will open in your browser at http://localhost:8501

## Structure

- **Data Exploration**: Dataset overview, morphology, phonology, semantics
- **Computational Experiments**: Model experiments and configurations
- **Results**: Model results and comparisons
- **Reporting**: Export and reporting functionality

## Requirements

Make sure streamlit is installed:
```bash
uv pip install streamlit
```
